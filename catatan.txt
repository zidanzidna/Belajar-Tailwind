    Pseudo class and Dark mode
- hover: untuk ketika cursor ada di element
- active: untuk button ketika dipencet
- focus: samping dari button ketika selesai dipencet
- group: digunakan untuk menyatukan element2nya
- selection: jika di selection textnya warnanya akan berbeda
- first-letter:  digunakan untuk mengubah huruf pertama dan didalamnya ada beberapa command(xl,float,my,mx,dll)
- placeholder: digunakan untuk membuat kotak inputan
- invalid: jika inputan tidak sesuai
- peer: sama seperti group namun digunakan jika sepadan
- invisible: digunakan jika ingin menghilangkan konten
- dark: mengubah element2nya ketika di dark mode
jika ingin membuat input, bungkus input dengan form, di labelnya for="(diisi dengan id yang ada di input, harus sama*)"
- scale : untuk memperbesar/ memperkecil element
- rotate : untuk memutar element
- translate : untuk menggeser element
- skew
- origin : menentukan sumbu element
TRANSITION and ANIMATION
- transition: buat element ada transisinya
- ease in,ease out: kecepatan transisi
- duration: kecepatan transisi
- delay: kecepatan animasi setelah terkena cursor
- animation: spin,pulse,bounce,ping
Jika, ingin diatur sesuai yg kamu inginkan maka harus di configurasi terlebih dahulu
LAYOUT
- container: digunakan untuk membatasi kiri kanan dari elementnya sesuai dengan breakpoint yg digunakan
- float: digunakan supaya tulisan mengelilingi gambar
- relative: digunakan untuk memberikan container/pembatas pada absolute
- absolute: digunakan untuk memindahkan element dengan bebas dalam relative
- flex: mengatur tata letaknya dengan flexible
- fixed: digunakan supaya elementnya tetap pada tempatnya dan selalu muncul di layar
- coloumns: digunakan untuk memberikan kolom pada gambar2 atau element2 
RESPONSIVE DESIGN
- tracking: buat jarak antar huruf
- objectcover: buat memotong gambar supaya sesuai dengan ukuran
